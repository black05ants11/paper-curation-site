このフォルダについて
====================
note（https://note.com/tarou_ymd_）に投稿した解説記事のサンプルを置くフォルダです。

入れるファイル
--------------
- 自分が書いた記事の Markdown ファイル（.md）
- ファイル名の例: 2024-01-gut-bacteria.md, 2024-03-sleep-longevity.md

用途
----
Claude に文体・構成・表現スタイルを学習させるための参照素材として使用します。
analyze_paper.py が生成する解説記事（article_md）の文章をこれらのサンプルに
近づけたい場合、プロンプトにこのフォルダの記事を few-shot として渡します。
